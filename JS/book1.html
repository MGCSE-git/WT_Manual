HTML5 - Basic Tags

Linking

One of the most important HTML5 features is the hyperlink, which provides links to other resources which may be of any type (word document, pdf, ppt, images, videos, etc.,). When a user clicks a hyperlink, it navigates to the particular resource. Any displayed element can act as a hyperlink. Web browsers typically underline text hyperlinks and color the text blue by default; example   google. The browser changes the color (to purple) of any text link once you’ve clicked the link. Example google. If the web server cannot locate a requested document, it returns an error indication to the web browser (known as a 404 error), and the browser displays a web page containing an error message. Links are created using anchor tag <a> along with the attribute href to specify the location of the resource.
